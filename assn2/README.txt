To run the program type the following command:

python3 allPairsShortestPath.py --algorithm [a b f] <dataset>

Chose argument a, b, or f to specify if you want to run both
algorithms, Bellman-Ford, or Floyd-Warshall. Lastly, for <dataset> 
choose the .txt file which contains the data you wish to use
as input for your algorithms.
