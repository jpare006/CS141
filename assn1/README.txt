To run the program type the following command:
python3 nearestNeighbor.py --algorithm [a b d] <dataset>
Choose argument a, b, or d to specify if you want to run
both alogirthms, the brute force algorithm, or the 
divide and conquer algorithm. Lastly, for <dataset> choose
the .txt file which contains the data you which to perform
the algorithms on.
